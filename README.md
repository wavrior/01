## Publish on GitHub Pages

1. Create a new repository (e.g. `portfolio`), upload all files to the root.
2. Repo **Settings → Pages → Source: Deploy from a branch → main / (root) → Save**.
3. Site goes live at `https://<username>.github.io/<repo>/` within a minute or two.
4. Custom domain (alexkiele.com): add the domain in the same Pages settings,
   then at your DNS provider point an `A`/`CNAME` record at GitHub Pages per
   the instructions GitHub shows you. Tick **Enforce HTTPS** once it verifies.
