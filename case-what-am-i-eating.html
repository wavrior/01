Overview

Working files are only a head start if you can work with them.

Unlike most localisation jobs, this one came with a package — PSDs and graphic assets from the original production. But the show is one of the most graphics-dense I've worked on, and the supplied files weren't structured for versioning: undocumented, inconsistently built, and never intended to be opened by another team. The job became equal parts localisation, file forensics and reconstruction.
